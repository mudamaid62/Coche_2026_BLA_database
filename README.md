### Coche at al., 2026 Antarctic beta-lactamases database

# Antarctic candidate beta-lactamase and related proteins from PBP-like and MBL-fold protein superfamilies database

- Coche_2026.faa is a multifasta file containing reference proteins from BLDB, true negative reference proteins, and Antarctic candidate beta-lactamase and related proteins sequences. For more information, check the preprint [bioRxiv](https://doi.org/10.1101/2025.09.30.679653 "A structure-guided pipeline to uncover the underexplored beta-lactamases from Antarctic and Subantarctic soil microbiota").
- Coche_2026_metadata.txt is a TSV file containing related information for each sequence in the database, including: Family, Class and Structural Clusters
